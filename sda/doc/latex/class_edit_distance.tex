Initialize Wagner-\/\+Fischer algorithm. 

For two strings\+: A of length n B of length m

We define D(i,j)\+: the edit distance between A\mbox{[}1..i\mbox{]} and B\mbox{[}1..j\mbox{]}, i.\+e. the edit distance between the first i characters in A and the first j characters in B

Thus, the edit distance between A and B is D(n,m)
