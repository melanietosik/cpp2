void EditDistance::init_matrix ( )
inlineprivate

Initialize Wagner-Fischer algorithm.

For two strings: A of length n B of length m

We define D(i,j): the edit distance between A[1..i] and B[1..j], i.e. the edit distance between the first i characters in A and the first j characters in B

Thus, the edit distance between A and B is D(n,m)

https://en.wikipedia.org/wiki/Wagner%E2%80%93Fischer_algorithm
